🚀 Sistem Fuzzy Mamdani

Prediksi Kegagalan Startup Berbasis Logika Fuzzy

📊 Tentang Metode Mamdani

Sistem ini menggunakan metode fuzzy Mamdani untuk memprediksi tingkat risiko kegagalan startup berdasarkan 4 faktor utama: usia perusahaan, jumlah pendiri, tingkat pendanaan, dan jumlah karyawan. Setiap variabel input dibagi menjadi kategori fuzzy (Rendah, Sedang, Tinggi) dengan fungsi keanggotaan trapezoidal.

Range: 0-20 tahun
Range: 1-10 orang
Range: 0-100 juta USD
Range: 1-500 orang
